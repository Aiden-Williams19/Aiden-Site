## Getting Started

1. **Clone or Download** this repository
2. **Add your photo**: 
   - Drag and drop your profile picture into this folder
   - Update the filename in `index.html` to match your image (currently set to `aiden.jpeg`)
3. **Open** `index.html` in your browser
4. **Start adding** your projects!

## Adding Projects

1. Click the "Add Project" button
2. Fill in the project details:
   - **Title**: Name of your project
   - **Description**: Brief overview
   - **Image URL**: Link to a project screenshot
   - **Live URL**: Link to the live site
   - **GitHub URL**: Link to the repository (optional)
   - **Tags**: Comma-separated tech stack (e.g., "React, Next.js, TypeScript")
   - **Type**: Personal or Client work
3. Click "Add Project" to save

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --navy-50: #f0f4f8;
    --navy-900: #102a43;
    /* ... etc */
}
```

### Contact Information
Update the links in `index.html`:
- Email address
- GitHub profile URL
- LinkedIn profile URL

### Content
Edit the text in `index.html`:
- Hero section title and subtitle
- About section paragraphs
- Footer text
